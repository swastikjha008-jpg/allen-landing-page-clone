# ALLEN Landing Page Clone

A responsive React landing page inspired by the ALLEN ASAT online courses page.

## Overview

This project is a frontend practice clone built with React and Vite. It recreates the structure of an educational landing page with a hero section, form card, benefit sections, course cards, process steps, and an interactive FAQ.

The goal of this project is to practice real-world layout building using simple React concepts and reusable UI sections.

## Features

- Responsive hero section
- Registration form UI
- Goal selection buttons
- Statistics section
- Benefits section
- Course cards
- Step-by-step process section
- Interactive FAQ using React state
- Clean visual layout

## Tech Stack

- React
- JavaScript
- CSS
- Vite

## Concepts Used

- React components
- `useState`
- array `map()`
- conditional rendering
- basic form state handling
- responsive layout design
